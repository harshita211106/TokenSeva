# TokenSeva
TokenSeva — A smart real-time queue management system that helps businesses manage customer queues digitally using QR-based token joining, live queue tracking, automated token updates, and admin dashboards for efficient customer handling.

---

## 🚀 Features

- Real-time queue updates using Socket.IO
- QR-based queue joining
- Live customer token tracking
- Admin dashboard for queue management
- Automated next-token calling system
- Estimated waiting time calculation
- REST API-based backend architecture
- MongoDB database integration
